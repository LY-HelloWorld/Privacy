# HomeInventory_web

Standalone static marketing page for HomeInventory.

## Files

- `index.html`: landing page with SEO meta tags, App Store CTA, structured data, FAQ, and product copy.
- `styles.css`: responsive styling for the landing page.
- `assets/app-icon.png`: local app icon asset used by the page and social previews.
- `robots.txt` and `sitemap.xml`: optional SEO helpers for GitHub Pages.

## Publish Notes

Before publishing, update `robots.txt` and `sitemap.xml` if your final GitHub Pages URL is not:

```text
https://ly-helloworld.github.io/HomeInventory_web/
```

The page has no build step. It can be opened directly from `index.html` or served by GitHub Pages.
